About MHCcombine


With MHCcombine we aim to enable you to simultaneously query several peptide-MHC class I binding prediction servers. After submitting a protein sequence and peptide features of interest this tool automatically sends your request to the selected servers for prediction and collects the output in a sortable file of comma-separated values (csv).

To cite MHCcombine please refer to: XXX.

The project MHCcombine started in 2013 by Stephanie Hoppe, Jan Winter, Christine Zeller and Angelika Riemer at the departement for Immunotherapy and -prevention in the German Cancer Research Center (DKFZ) Heidelberg. The idea was to develop a handy tool to simplify the combination of different prediction server results for easy comparison.
