The MHCcombine WebApp combines up to 12 different MHC class-I binding prediction methods from several prediction servers.

With MHCcombine you can simultaneously query several peptide-MHC class-I binding prediction servers. After submitting a protein sequence and your required peptide specification this tool automatically sends your request to the selected servers for prediction and collects the output in a sortable file of comma-separated values (csv).
